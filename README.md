# Mon-Portfolio - Davian Mbayi

Bienvenue sur mon portfolio web personnel, une vitrine de mon parcours, de mes projets et de mes compétences en développement web.
Ce site a été entièrement conçu en HTML, CSS et JavaScript, dans une approche moderne et responsive.

Lien du projet : https://monportfolio-davianmbayi.netlify.app/

Le site présente :
1. Mes projets récents

2. Mon parcours de formation

3. Mes certifications

4. Une section À propos avec mes compétences techniques

5. Mes coordonnées pour me contacter ou consulter mes profils professionnels

A. Technologies utilisées

HTML5 – Structure du site

CSS3 – Mise en page et animations

JavaScript – Interactions et effets dynamiques (scroll, animations, date automatique)

Git / GitHub – Gestion de version et hébergement

B. Structure du projet
.
├── index.html             # Page principale du portfolio

├── style.css              # Feuille de style principale

├── image/                 # Images de profil et illustrations

├── Projet/                # Projets réalisés (ex. Moyenne.html)

├── Mon CV/                # CV au format PDF

└── certifications/        # Dossiers contenant les certificats (PDF/PNG)

C. Fonctionnalités notables :

1. Design responsive adapté aux mobiles et tablettes

2. Animations au défilement via Intersection Observer

3. Mise à jour automatique de l’année dans le pied de page

4. Navigation fluide par ancres internes

5. Effet visuel moderne avec ombres, dégradés et transitions
